#### Fórmula da Distância

$D = \sqrt[2]{(x2-x1)^2 + (y2-y1)^2}$

$R_1 = \text{Raio do círculo 1}$  
$R_2 = \text{Raio do círculo 2}$

$Colisão: R_1 + R_2 < D$
